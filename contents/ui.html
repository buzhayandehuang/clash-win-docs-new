# 界面组成

CFW 整体界面由如下部分组成：

  • General：用于修改常规设置，如系统代理和自启动
  • Proxies：用于修改代理模式及策略节点
  • Profiles：用于管理多个配置文件
  • Logs：用于查看网络请求记录
  • Connections：用于查看/管理当前网络连接
  • Settings: 设置CFW的相关配置
  • Feedback：文档相关及推广
